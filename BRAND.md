# EchoBrief Brand Guidelines — Warm Ember (Option B)

## Colors
### Primary
- Orange 500: #F97316 (Primary brand)
- Amber 500: #F59E0B (Gradient end)
- Orange 400: #FB923C (Links, hover)
- Orange 900: #7C2D12 (Dark accent)

### Neutrals (Stone)
- Stone 50: #FAFAF9 (Light bg / text)
- Stone 200: #E7E5E4 (Borders)
- Stone 400: #A8A29E (Secondary text)
- Stone 800: #292524 (Cards / dark UI)
- Stone 950: #0C0A09 (Dark background)
- Stone 900: #1C1917 (Dark surface)

### Semantic
- Green 500: #22C55E (Success / recording)
- Blue 500: #3B82F6 (Info / processing)
- Red 500: #EF4444 (Error / stop)
- Purple 500: #A855F7 (AI / insights)

### Gradient
- `background: linear-gradient(135deg, #F97316, #F59E0B);`
- Use sparingly: hero CTAs, active states, app icon
- NEVER for body text, large surfaces, error/warning/destructive actions
- 4px gradient bar as signature detail on cards and section headers

## Typography
- Display: Outfit 600 (32-48px) — Hero headlines
- H1: Outfit 600 (24-28px) — Page titles
- H2: Outfit 500 (18-20px) — Card titles
- Body: DM Sans 400 (14-16px) — Paragraphs
- Body bold: DM Sans 500 (14-16px) — Labels
- Caption: DM Sans 400 (12px) — Metadata
- Code: JetBrains Mono 400 (13px) — Transcripts, API
- Indic: Noto Sans [script] 400/500 (min 16px)

## Logo
- Logomark: concentric sound ripples (3 rings)
- Dark bg: 'echo' white, 'brief' Orange 400
- Light bg: 'echo' Stone 900, 'brief' Orange 600
- Clear space: diameter of center dot on all sides
- Min size: 120px wide (full lockup), 24px (mark alone)

## Icons
- Library: Lucide React, 24px, 1.5px stroke
- Recording: pulsing green dot (#22C55E), 1.5s animation
- Stop: Red 500 (#EF4444)
- NEVER use orange for recording states

## Buttons
- Primary: gradient bg, white text, 12px radius, font-weight 500
- Ghost: transparent bg, Orange 500 text, 1px orange border on hover

## Voice & Tone
- Conversational, not corporate
- Outcome-first (lead with what user gets)
- Confident, not arrogant (no exclamation marks or superlatives)
- India-aware (WhatsApp, INR, name languages specifically)
- NEVER write "AI-powered", "cutting-edge", "next-gen"

## Don'ts
- No orange for large background fills
- No orange + red in close proximity
- No stretching/rotating/effects on logo
- No Outfit for body text (headings only)
- No gradient for error/warning/destructive actions
